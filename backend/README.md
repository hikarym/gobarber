## :rocket: Acerca do projeto

Aplicação que permite gerehciar agendamentos com o cavaleleiro.

### Inicializar a APP

```
$ yarn dev:server
```

### Executar a APP

```
$ yarn dev:server
```

### Tecnologias usadas

- NodeJS junto ao TypeScript
- Express
- Postgres
- Uso de banco de dados com o TypeORM
- Envio de arquivos com o Multer
